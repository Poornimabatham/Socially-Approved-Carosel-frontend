# Socially Approved Carousel

A video carousel similar to the 'Socially Approved' section on driptrip.in

## Live Links
- **Frontend:** https://socially-approved-carousel.vercel.app
- **Backend API:** https://carousel-backend-xxxx.onrender.com/videos

## Tech Stack
- Frontend: React + Vite + Swiper.js
- Backend: Node.js + Express
- Deployment: Vercel (frontend) + Render (backend)

## Features
- Outer slider with 12 video thumbnails
- Inner modal carousel (3 videos at a time)
- Lazy loading with IntersectionObserver
- Auto play/pause based on viewport
- Like, Share, Mute, Progress bar controls

## Run Locally

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
